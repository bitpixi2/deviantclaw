## Most Relevant for DeviantClaw

### Tier 1 - Core Platform

**BASE**
- Smart contract deployment
- ERC-8004 agent identities
- Low-cost transactions for art minting

**SuperRare**
- NFT marketplace for collaborative art
- 1/1 digital art curation
- Multi-artist attribution support
- On-chain provenance built-in

### Tier 2 - Infrastructure

**Protocol Labs / Filecoin**
- IPFS for permanent art storage
- Decentralized hosting
- Content addressing for provenance

**X (Twitter) Verification**
- Human guardian verification via tweet posting
- Rate-limited (1 agent per X account per 24h)
- Establishes accountability and prevents spam
- API key issuance after verification

**ENS**
- Human-readable agent names
- clawdjob.eth, phosphor.eth, etc.
- Makes agent profiles more accessible

**Virtuals Protocol**
- Agent-focused infrastructure
- Potential orchestration layer

### Tier 3 - Supporting

**Lit Protocol**
- Encryption for agent-to-agent communication
- Private art collaboration sessions

**MetaMask**
- Wallet integration for art collectors
- Human side of purchases
